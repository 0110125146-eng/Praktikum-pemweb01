    Tugas 3: FAQ Interaktif

    1. Apa itu JavaScript?
    Bahasa pemrograman tingkat tinggi yang digunakan untuk membuat halaman web interaktif, dapat dijalankan di sisi klien maupun server.
    2. Apa perbedaan DOM dan jQuery?
    DOM (Document Object Model) adalah antarmuka pemrograman untuk dokumen HTML/XML yang memungkinkan akses dan manipulasi elemen. jQuery adalah library JavaScript yang menyederhanakan operasi DOM, event handling, dan animasi.
    3. Bagaimana cara menambahkan event click pada elemen?
    Pada vanilla JavaScript bisa menggunakan addEventListener(), sedangkan pada jQuery menggunakan metode .click() atau .on('click', function(){...}).
    4. Apa itu method chaining di jQuery?
    Teknik untuk menulis beberapa metode jQuery secara berurutan pada satu elemen tanpa perlu menyimpan referensi elemen tersebut ke variabel terpisah, misalnya $(elemen).addClass('x').fadeIn().slideDown().
    5. Apa fungsi dari metode .toggle() di jQuery?
    Metode ini digunakan untuk menyembunyikan atau menampilkan elemen secara bergantian. Jika elemen sedang ditampilkan, maka akan disembunyikan, dan sebaliknya.
